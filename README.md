# aerospike-docker-compose

Two node Aerospike cluster with monitoring. (prometheus and grafana)


1. Run 
   ```sh
   docker compose up
   ```
2. Grafana  User:admin pass:admin
   ```sh
   http://localhost:4000
   ```
